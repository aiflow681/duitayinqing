【游戏名称】
堆塔游戏 - 引擎导出版

【游戏介绍】
这是一款使用游戏引擎导出的休闲益智类小游戏。游戏中，你需要控制摇摆的方块，在合适的时机放下，将方块堆叠成高塔。堆得越高，分数越高！本版本采用简洁清爽的视觉风格，天空蓝配色，现代扁平化设计。

【玩法说明】
游戏中，方块会在空中左右摇摆，你需要在合适的时机放下方块。

规则：
1. 方块会在钩子上左右摇摆，点击屏幕放下方块。
2. 方块需要准确堆叠在下方的方块上，超出部分会被切掉。
3. 连续完美堆叠可以获得额外分数。
4. 方块越堆越小，失败3次游戏结束。

操作：
点击屏幕或按空格键 - 放下方块。
尽量让方块完美对齐，获得更高分数！

目标：
堆叠尽可能多的方块，挑战最高分！

Tips：游戏结束会记录你的最高分数和堆叠层数。
完美堆叠可以让方块不变小，甚至变大！掌握节奏是关键。

【技术介绍】
主要语言是JavaScript，使用了Canvas 2D渲染技术，Zepto.js轻量级框架，物理引擎等。
