【游戏名称】
堆塔游戏

【游戏介绍】
这是一款H5原生开发的休闲益智类小游戏。游戏中，你需要通过精准的点击时机，将不断左右移动的方块一层层堆叠起来，挑战建造最高的塔楼。游戏考验你的反应力和节奏感，随着层数增加，难度也会逐渐提升！

【玩法说明】
游戏中，方块会在塔楼上方左右移动，你需要在合适的时机点击屏幕，让方块准确落在下方的塔楼上。

障碍：
1. 方块会不断左右移动，移动速度会随着层数增加而加快。
2. 如果放置不够精准，超出下方方块的部分会被切除掉落，下一块方块会变小。
3. 方块越小，游戏难度越高，更容易失误。

操作：
点击屏幕任意位置（或按空格键）即可放下移动中的方块。
游戏开始前，点击屏幕或按空格键开始游戏。

目标：
如果方块完全未能放置在塔楼上（完全错开），游戏就会结束。尽可能堆叠更多层，挑战最高分！

Tips：游戏结束会显示你的最终得分。
点击"再玩一次"按钮，刷新你的最高分记录！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术，纯原生实现无需任何框架。
